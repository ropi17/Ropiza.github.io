# HabitForge - Modern Habit Tracker

Aplikasi pemantau kebiasaan (habit tracker) modern dengan fitur gamifikasi, dark mode, dan tips harian.

## Cara Menggunakan

1.  Buka folder ini.
2.  Klik dua kali pada file `index.html`.
3.  Aplikasi akan terbuka di browser Anda. Tidak perlu instalasi npm atau server khusus!

## Fitur Utama

-   **Dashboard Interaktif**: Checklist harian yang memuaskan.
-   **Gamifikasi**: Dapatkan XP dan naik level setiap kali menyelesaikan habit.
-   **Streak & Statistik**: Pantau konsistensi Anda dengan grafik visual.
-   **Smart Categorization**: Kelompokkan habit (Kesehatan, Produktivitas, dll).
-   **Daily Wisdom**: Tips harian dari metode Atomic Habits.
-   **Dark Mode**: Mendukung mode gelap yang nyaman di mata.
-   **Data Persisten**: Data tersimpan otomatis di browser (LocalStorage), jadi tidak akan hilang saat direfresh.

## Teknologi

-   HTML5
-   Tailwind CSS (via CDN)
-   Vanilla JavaScript
-   Lucide Icons
-   Font Google (Inter)
